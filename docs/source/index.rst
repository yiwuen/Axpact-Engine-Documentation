Axpact Engine Documentation
===================================

Welcome to the official documentation of Axpact Engine! Now, if you don't know what Axpact Engine is, it's a 2D and 3D game engine, built specifically for multiplayer games though you can use this game engine to create any types of game your like, for example, singleplayer games. With the help of this documentation, you can get started quickly with Axpact Engine and creating your first game.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
