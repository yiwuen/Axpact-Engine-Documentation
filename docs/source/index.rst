Axpact Engine Documentation
===================================

Welcome to the official documentation of Axpact Engine! Now if you don't know what Axpact Engine is, Axpact Engine is a free and open source 2D and 3D game engine built specifically for multiplayer games though you can create any types of games with this engine like singleplayer games. With the help of this documentation, you can quickly and easily get started with this game engine in no time and build your first game.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
